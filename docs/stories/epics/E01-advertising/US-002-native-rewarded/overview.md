# Overview

## Current Behavior

Native and rewarded IDs exist as future placeholders but no code requests or renders those formats.

## Target Behavior

The full transaction list renders a small native template after every eight transactions. Ranking
offers an explicit rewarded action that reveals a savings insight only after reward confirmation.

## Affected Users

- Android and iOS users browsing transaction history or savings ranking.

## Affected Product Docs

- `docs/product/advertising.md`

## Non-Goals

- Automatic rewarded display, rewards that modify balances/rank, app-open ads, or SSV-backed value.
